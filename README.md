# Job Portal React

Full-featured Job Portal web application built with React.

## Features
- User authentication
- Protected routes
- Add, edit, delete job postings
- Search jobs
- Apply for jobs
- Responsive UI

## Tech Stack
- React
- Vite
- JSON Server (for mock backend)
- CSS

## Installation

1. Clone the repository
2. Install dependencies
   npm install
3. Start development server
   npm run dev
